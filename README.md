## Game Description

In this game, the player controls a bird that continuously falls due to gravity. By pressing the spacebar or clicking the bird jumps upward.Pipes move from right to left across the screen and the player must guide the bird through the gaps between the pipes without colliding. If the bird hits a pipe or the ground, the game ends.Each time the bird successfully passes through a set of pipes  the players score increases. The goal is to survive as long as possible and achieve the highest score.
